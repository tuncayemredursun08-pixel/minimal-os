# Prebuilt Images

Prebuilt ISO images are available in the [Releases](https://github.com/tuncayemredursun08-pixel/minimal-os/releases) section.\n\n## Running Prebuilt Image\n\n### Using QEMU\n```bash\nqemu-system-x86_64 -cdrom minimal-os.iso -m 512M -serial stdio\n```\n\n### Using VirtualBox\n1. Create a new VM (64-bit, 512MB RAM)\n2. Mount the ISO as the boot CD\n3. Start the VM\n\n### On Real Hardware\n1. Write ISO to USB:\n   ```bash\n   dd if=minimal-os.iso of=/dev/sdX bs=4M && sync\n   ```\n2. Boot from USB\n\n## Image Details\n\n- **Size**: ~5MB (compressed)\n- **Resolution**: 1024×768 (framebuffer)\n- **RAM**: Minimum 256MB\n- **Features**:\n  - GRUB bootloader\n  - Minimal x86-64 kernel\n  - Custom overlapping window manager\n  - VGA text console\n  - Framebuffer graphics\n\n## System Info\n\nOnce booted:\n- Press `ALT+F2` for system info\n- Press `ESC` to exit\n- Mouse clicks focus windows\n- Click window title bars to drag\n\n## Building from Source\n\nIf you want to build the image yourself:\n```bash\ngit clone https://github.com/tuncayemredursun08-pixel/minimal-os\ncd minimal-os\nmake clean\nmake\n```\n\nThe ISO will be created as `minimal-os.iso`\n"
